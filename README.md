# Lavei

App mobile nativo para intermediação de serviços de lavagem automotiva, construído com **React Native + Expo** e backend **Express.js**.

## Começo Rápido

### 1. Baixe o Expo Go
Instale o app "Expo Go" no seu celular (iOS ou Android).

### 2. Inicie o app
```bash
npm start
```

### 3. Escaneie o QR code
Use o Expo Go para escanear o código que aparecer no terminal.

## Features

✅ Login com Replit Auth  
✅ Navegação por abas (Início, Histórico, Conta)  
✅ Notificações push em background  
✅ Design mobile-first com branding Lavei  

## Tech Stack

- **Frontend**: React Native + Expo
- **Backend**: Express.js + TypeScript
- **Database**: PostgreSQL (Neon) + Drizzle ORM
- **Auth**: Replit Auth (OpenID Connect)

## Scripts

```bash
npm start              # Inicia Expo
npm run server         # Backend API (porta 5000)
npm run android        # Android
npm run ios            # iOS
npm run db:push        # Sincronizar banco
npm run db:studio      # Drizzle Studio
```
